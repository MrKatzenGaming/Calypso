# Calypso

## Server

### Prerequisites

`pip install websockets>=14.0`

### Running the server

`python3 server/run.py`

## Building

Follow the instructions listed in [LibHakkun's README](https://github.com/fruityloops1/LibHakkun?tab=readme-ov-file#setup).
